# CalendarApp

CalendarApp is an Angular application that allows users to manage appointments and view them in a calendar format. This project leverages Angular Material and Angular CDK for a streamlined development process, providing a simple yet powerful calendar application.

## Features

1. **Form to Add Calendar Appointment**: Users can add new appointments using a form.
2. **Ability to Delete Appointments**: Users can delete existing appointments.
3. **Ability to Move Appointments**: Users can move appointments to different dates or times using drag-and-drop functionality.
4. **Calendar View**: The application displays appointments in a calendar format using `ngFor`.
5. **Drag & Drop**: Implemented using Angular CDK for an intuitive user experience.
6. **Dependency Injection**: Proper use of Angular's dependency injection system.
7. **Lazy Loading**: Modules and routes are lazy-loaded for performance optimization.
8. **Routing**: Includes routing with router-outlets for navigation.
9. **Angular Forms**: Utilizes reactive forms with value changes, validators, etc.
11. **Standalone Components and Shared Modules**: Application is modularized for better maintainability.

## Requirements

- **Angular**: Version 16.0.2
- **Angular Material**: For UI components
- **Angular CDK**: For drag-and-drop functionality

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Build

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory. Use `ng build --prod` for a production build.

## Running unit tests

Run `ng test` to execute the unit tests via [Karma](https://karma-runner.github.io). This will ensure that all unit tests pass and the application behaves as expected.
